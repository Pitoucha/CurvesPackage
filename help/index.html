Nucleic Anlaysor

Nucleic Analysor is a plugin who can used conjointly with VMD. It offers various analysis running on double helix DNA and G-Quadruplex DNA

Get started

Prerequisite

In order to use our plugin it's required that you already have this ?dependencies? installed

  • A Linux or Mac distribution : at the time of the creation of this website, Nucleic Analysor isn't compatible with Windows, we are working on finding a solution in the near future

  • VMD (1.9.3 version or later)
    https://www.ks.uiuc.edu/Development/Download/download.cgi?PackageName=VMD


  • GnuPlot Package (for creating Histograms)
    http://www.gnuplot.info/download.html

Installation

To install Nucleic Analysor, first you need to dowlad the package here
https://github.com/Pitoucha/NucleicAnalysor.git
Once dowloaded, verify that it contains 2 folders named GNU_Script and Help, and two files named NucleicAnalysor.tcl and pkgIndex.tcl. Copy the folderNucleicAnalysorto VMD TCL plugin directory$VMDDIR/plugins/noarch/tcl/

Then insert the following line into$VMDDIR/scripts/vmd/loadplugins.tclat line 152 :


vmd_install_extension NucleicAnalysor nucleicanalysor_tk "Analysis/Nucleic Analysor"
